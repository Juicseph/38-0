Google Sheet Setup

  1. Put your recruit data into one Google Sheet tab called Recruits.
  2. Paste the Apps Script below into Extensions → Apps Script.
  3. Deploy it as a Web App with access set to Anyone with the link.
  4. Paste that web app URL into WEB_APP_URL in this HTML.
  5. When the sheet changes, hit refresh here.
